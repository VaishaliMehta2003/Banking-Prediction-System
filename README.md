# 🏦 Banking Prediction System

An end-to-end Banking Analytics & Machine Learning project that predicts **Loan Risk**, **Credit Card Adoption**, and performs **Customer Segmentation** using real-world banking data. The project integrates **SQL, Python, Machine Learning, Power BI, and Streamlit** into a complete analytics pipeline.

# 📌 Project Overview

This project demonstrates a complete Banking Analytics workflow starting from data extraction and preprocessing to predictive modeling and deployment.

The system helps banks:

- Predict customers with high loan default risk.
- Identify customers likely to adopt credit cards.
- Segment customers based on banking behavior.
- Provide an interactive prediction interface using Streamlit.

---

# 🎯 Objectives

- Clean and preprocess banking data.
- Perform Exploratory Data Analysis (EDA).
- Build predictive machine learning models.
- Deploy the models using Streamlit.
- Create an end-to-end analytics solution suitable for business decision-making.

# 🤖 Machine Learning Models

## 1️⃣ Loan Risk Prediction

**Objective**

Predict whether a customer is at risk of loan default.

**Algorithm Used**

- Random Forest Classifier

**Output**

- Low Risk
- High Risk

---

## 2️⃣ Credit Card Adoption Prediction

**Objective**

Predict whether a customer is likely to adopt a credit card.

**Algorithm Used**

- Random Forest Classifier

**Output**

- Likely to Adopt
- Not Likely to Adopt

---

## 3️⃣ Customer Segmentation

**Objective**

Group customers based on their banking behavior.

**Algorithm Used**

- K-Means Clustering

**Output**

- Customer Cluster 0
- Customer Cluster 1
- Customer Cluster 2
